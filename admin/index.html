Inserisci l'URL Web App di Google Apps Script. Lo stesso endpoint deve accettare POST dalla mobile app e GET dalla dashboard desktop. Consiglio fogli separati: Interventi, Segnalazioni, RegistroLegionella, Scorte, Turni, Compliance, ConfermeTeam.

GET atteso: { shifts:[], logs:[], issues:[], legionellaRegister:[], reservations:[] }
POST mobile consigliato: { type:'intervento'|'segnalazione'|'legionella'|'scorte', payload:{...} }
